# StyleSync Pro - Sistema de Gestión de Citas

Este proyecto es una plataforma web completa para la administración de un estudio de estética (StyleSync Pro), que incluye una landing page para clientes y un panel de administración avanzado para la gestión de citas y especialistas.

## 🚀 Funcionalidades Principales

### 📅 Agenda Inteligente (Workflow Multinivel)
- **Bandeja de Entradas**: Recepción de citas pendientes por confirmar asistencia.
- **Bandeja En Proceso**: Control de clientes que ya están en el estudio siendo atendidos.
- **Bandeja Aplazadas**: Gestión de reprogramaciones con validación de conflictos de horario.
- **Liquidación**: Botones específicos de "PAGÓ" o "NO PAGÓ" para mover citas al historial con estado financiero correcto.

### 👩‍🎨 Gestión de Especialistas
- **Reportes Detallados**: Generación de reportes de servicios y ganancias por día o mes.
- **Cálculo Automático**: División de ganancias entre el estudio (60%) y la especialista (40%).
- **Exportación a PDF**: Reportes profesionales descargables con un solo clic.

### 🛠️ Administración de Catálogos
- **Servicios**: Subida de fotos, descripción y precios.
- **Galería**: Gestión visual de trabajos realizados.
- **Categorías**: Organización por acordeones y menús desplegables con animaciones suaves.

### 📱 Experiencia de Usuario (UI/UX)
- **Diseño Premium**: Paleta de colores Dark Pink/Rose, efectos de cristal (Glassmorphism) y animaciones fluidas.
- **Responsivo**: Adaptado para su uso en computadoras y dispositivos móviles.
- **Soporte de Teclado**: Atajos con Enter y Escape para diálogos rápidos.

## 🛠️ Tecnologías Usadas
- **Frontend**: HTML5, Vanilla CSS3, JavaScript (ES6+).
- **Backend**: Integración con Firebase (en desarrollo/opcional) y almacenamiento local (LocalStorage) para persistencia rápida.
- **Librerías**: FontAwesome (iconos), JSPDF (generación de documentos).

## 📦 Instalación y Uso
1. Clona este repositorio.
2. Abre `index.html` para ver la web del cliente.
3. Abre `admin.html` para acceder al panel de control (Usuario: `admin`, Clave: `123456`).
